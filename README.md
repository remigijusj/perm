perm
=========
Permutations are elements of a symmetric group on integers 0 .. n-1.

This package provides simple calculations with permutations:
* identity
* random
* inverse
* composition
* power
* conjugate
* signature
* sign
* order
* equality tests
* conversion from/to cyclic form

INSTALL
-------
	go get github.com/remigijusj/perm
